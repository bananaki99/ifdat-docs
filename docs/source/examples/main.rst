Εισαγωγή στοιχείων
==================

Η προτεινόμενη ροή εργασίας είναι πρώτα τα λογιστικά και σειριακά όταν
συναντώνται αναγνωριστικοί κωδικοί ελέγχεται αν χρειάζεται να συμπληρωθούν τα
στοιχεία αναφοράς τους.  Αυτό ισχύει και για τα ίδια στοιχεία αναφοράς αλλά και
τα στοιχεία αναφοράς αντισυμβαλλομένων.
