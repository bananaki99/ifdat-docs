Yποβολή Τ0 (περίοδος αναφοράς Ιανουάριος 2022)
==============================================


Εξελίξεις κατά τη διάρκεια του Ιανουαρίου του 2022
--------------------------------------------------
#. Σύσταση ΑΕΔΑΚ με αναγνωριστικό κωδικό EGRMFMC000 -> Συμπλήρωση πινάκων REF.MNGMNT, REF.MNGMNT_DYNMC
#. Σύσταση τεσσάρων ΟΣΕΚΑ με αναγνωριστικό κωδικό  -> Συμπλήρωση πινάκων REF.FND, REF.FND_DYNMC
#. Έκδοση κατηγοριών μεριδίων -> Συμπλήρωση πινάκων REF.SELF_SHR, REF.SELF_SHR_DYNMC
#. Ορισμός προσώπου επικοινωνίας -> Συμπλήρωση πινάκων REF.CNTCT, REF.CNTCT_T_RA
#. Άνοιγμα καταθετικών λογαριασμών -> Συμπλήρωση πίνακα REF.DPST
#. Λογιστικά στοιχεία -> Συμπλήρωση πινάκων ACC.DPST, ACC.HLDR, ACC.AGG
