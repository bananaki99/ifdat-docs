Στοιχεία Αναφοράς (REF)
=======================

Τα στοιχεία αναφοράς αναγγέλονται μόνο όταν υπάρχουν νέα στοιχεία που δεν έχουν
υποβληθεί στο παρελθόν καθως και όταν δημιουργείται ανάγκη για μεταβολλές ή
διορθώσεις στοιχείων που έχουν υποβληθεί στο παρελθόν.
