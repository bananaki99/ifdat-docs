A. Κατηγορία Πινάκων με Στοιχεία Αναφοράς (REF)
===============================================

Οι πίνακες αυτοί περιλαμβάνουν στοιχεία αναφοράς και περιέχουν διάφορα
πληροφοριακά στοιχεία που είτε παραμένουν σταθερά είτε δεν μεταβάλλονται συχνά.
Τα στοιχεία αναφοράς συμπληρώνονται εφόσον υπάρχουν νέα στοιχεία, μεταβολές ή
διορθώσεις προηγούμενων στοιχείων αλλά και στη περίπτωση εμπλουτισμού ιστορικών
στοιχείων.  Για κάθε πίνακα όλες οι μετρήσεις είναι προαιρετικές.  Θα πρέπει
όμως να συμπληρωθεί τουλάχιστον μία μέτρηση για κάθε παρατήρηση.
