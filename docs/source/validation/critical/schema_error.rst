SCHEMA ERRORS
=============

Για κάθε λάθος δίδονται οι ακόλουθες πληροφορίες:

Ονομασία αρχείου (filename)
    Η ονομασία του αρχείου από όπου προήλθε το λάθος.

Τοποθεσία (location)
    Η τοποθεσία εντός του αρχείου που προήλθε το λάθος.

Περιγραφή (message)
    Περιγραφή του λάθους.

domain
    The error domain

type
    The error type

level
    The error level

line
    The error line

column
    The error column
