Εισαγωγή κύριου όγκου των στοιχείων
===================================

Η προτεινόμενη ροή εργασίας είναι πρώτα τα λογιστικά και σειριακά όταν συναντώνται αναγνωριστικοί κωδικοί ελέγχεται αν χρειάζεται να συμπληρωθούν τα στοιχεία αναφοράς τους.  Αυτό ισχύει και για τα ίδια στοιχεία αναφοράς αλλά και τα στοιχεία αναφοράς αντισυμαλλόμνων.  Δεν ισχύει μόνο για τα στοιχεία αναφοράς νέων επενδυτικών οργανισμών που ανακοινώνονται τις δύο πρώτες εργασιμες μέρες του μήνα και συμπληρωνονται οι πίνακες FND και FND_DYNMC.
