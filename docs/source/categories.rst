====================

Για κάθε από τις έξι κατηγορίες δεδομένων του IFDat δίδονται: 

    * η ονομασία και σε παρένθεση η κωδική ονομασία της κατηγορίας δεδομένων

    * Ονομασία, κωδική ονομασία σε παρένθεση (σε περίπτωση που διαφέρει από την ονομασία) και περιγραφή των διαστάσεων του πίνακα.

    * Ονομασία, κωδική ονομασία σε παρένθεση (σε περίπτωση που διαφέρει από την ονομασία) και περιγραφή των μεταβλητών του πίνακα.
